# WirtualnaUczelnia

W pierwszej kolejności, aby program uruchomić z poziomu swojego Visual Studio 2022 należy:

  1. We wierszu poleceń pakietów NuGet wpisać: Add-Migration InitialCreate
  2. Kolejna komenda: Update-Database

Po stworzeniu swojej instancji bazy danych możemy uruchomić aplikację (F5). Otworzy się przeglądarką z adresem np. https://localhost:7104/ lub innym portem.

Z tego miejsca możemy przeglądać Wirtualną Uczelnię.

Budynki i miejsca, a także przejścia dodajemy po wejściu do panelu administratora. Wchodzimy w link logowania znajdujący się w prawym dolnym rogu.
Wpisujemy następujące dane:
  admin@wlodkowic.pl
  haslo1234PL!?

Po tym na górnym menu pojawi się czerwona belka z listą rozwijaną, na której jest menu do edycji Budynków, Lokacji oraz Przejść.
